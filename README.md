# 📱 Flipkart Mobile Sales Analysis Dashboard (2020–2024) using Excel
The Flipkart Mobile Sales Dashboard is a data analytics project developed using Microsoft Excel with the objective of analyzing and visualizing mobile phone sales data in a structured and meaningful way. This project focuses on transforming raw transactional data into an interactive dashboard that supports data-driven decision-making.

## 📌 Project Overview
This project is an interactive Mobile Sales Analysis Dashboard created using Microsoft Excel. It analyzes mobile sales data from Flipkart over the period 2020–2024.

The dashboard provides insights into brand performance, sales trends, customer ratings, and overall market behavior.

---

## 🎯 Objectives
- Analyze mobile sales trends over time (2020–2024)
- Compare performance of different brands (POCO, Realme, Samsung, etc.)
- Track customer ratings and product performance
- Identify top-selling models and peak sales periods

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

- Data Cleaning & Transformation

---

## 📊 Key Features
- Brand-wise filtering (POCO, Realme, Samsung)
- Sales trend analysis (year-wise)
- Customer rating insights
- Top-performing mobile models
- Interactive and dynamic dashboard design
